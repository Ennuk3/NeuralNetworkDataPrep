plot_pred_func

This function uses the multi-index array of all of the models (see read function) as an input and returns comparison plots of full Charpy curves between all of the input models.

Parameters
----------
dataframe : DataFrame
     multi-index DataFrame consisting of all of the models and compositional profiles
columns : list
     column names for the individual datapoints
path : str
     directory path to save the plots

Returns
-------
model_list : list
     list of all of the models
comp_list : list
     list of all of the compositional profiles pred_list : list
     list of all of the predicted variables
